## Instalacion :warning:

### :running: Ejecutar Local

Clonar el proyecto

```sh
- git clone https://github.com/jeroalderete/UX-Snowboards
```

Ir a la carpeta del proyecto

```bash
  cd UX-Snowboards
```

Start server

```bash
  run Live Server On Vs Code
```
